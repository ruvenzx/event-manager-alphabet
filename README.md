# event-manager-alphabet
AlphaBet Backend Home Excercise

## 1. Prerequisites 
### Docker installed
### Python 3.6+

## 2. Howto use
### cd to folder
### run 'docker-compose up -d --build'
### go to http://localhost:8008/docs for api docs
### debug using 'docker-compose logs -f web'


## 3. Tests
### use pytest
### cd to folder and run 'pytest'
